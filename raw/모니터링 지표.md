투자 의사결정을 위한 핵심 지표 모음

**관련 문서**: [[API 데이터 수집 가이드]] - 각 지표를 API로 자동 수집하는 방법

## 1. 유동성 지표

시장에 자금이 얼마나 풍부한지, 돈의 흐름이 주식시장에 유리한지 판단하는 지표

### 1.1 미국 국채 10년물 금리
- **의미**: 글로벌 무위험 수익률의 기준
- **활용**: 금리 상승 시 주식 매력도 하락 (채권으로 자금 이동)
- **모니터링**: Fed 정책, 인플레이션 기대, 경기 전망 반영

### 1.2 달러 환율
- **의미**: 달러 강세/약세에 따른 글로벌 유동성 변화
- **활용**: 달러 약세 시 신흥국 및 원자재 강세, 수출주 수혜
- **모니터링**: DXY(달러인덱스), 원/달러 환율

### 1.3 금융상황지수 (FCI, Financial Conditions Index)

**의미**: 금융시장의 전반적인 유동성과 신용 환경을 하나의 숫자로 나타낸 종합 지수

**구성 요소**:
- 단기/장기 금리 (국채, 회사채)
- 환율 (달러 강도)
- 주식 가격
- 신용 스프레드 (BBB-Treasury, High Yield Spread)
- 은행 대출 조건

**계산 방법**:
각 기관마다 방식이 다르지만, 기본 구조는:
```
FCI = w1×금리변화 + w2×신용스프레드 + w3×환율변화 + w4×주가변화 + ...
(w는 각 요소의 가중치)
```

**예시 - 블룸버그 미국 FCI**:
- 기준점: 0 (중립)
- 양수(+): 금융여건 긴축 → 자금 조달 어려움 → 주식 약세
- 음수(-): 금융여건 완화 → 자금 조달 쉬움 → 주식 강세

**실제 활용**:
- 2023년 초: FCI가 0.5에서 -0.3으로 하락 → 연준의 긴축에도 불구하고 금융여건 완화 → S&P 500 상승
- 2022년 10월: FCI +1.2(긴축) → 주식 바닥권 신호 → 이후 반등

**확인 방법**:
1. 블룸버그 터미널: BFCIUS Index
2. 골드만삭스 FCI: 공개 리포트
3. 시카고 연은 FCI: https://www.chicagofed.org/
4. 대안: 10년물 금리 + HY 스프레드 직접 모니터링

## 2. 차트(심리) 지표

기술적 분석과 시장 참여자들의 심리를 파악하는 지표

### 2.1 이동평균선
- **의미**: 일정 기간 주가의 평균선 (5일, 20일, 60일, 120일, 200일)
- **활용**: 주가가 이평선 위에 있으면 상승추세, 아래면 하락추세
- **모니터링**: 골든크로스(단기선이 장기선 상향돌파), 데드크로스(하향돌파)

### 2.2 이평선상회 종목 비중
- **의미**: 시장 전체 종목 중 특정 이평선(예: 20일선) 위에 있는 종목 비율
- **활용**: 70% 이상이면 과열, 30% 이하면 침체 신호
- **모니터링**: 시장 전반의 강도와 참여 종목 확산 정도 파악

### 2.3 이격도 (50일/200일)

**의미**: 현재 주가가 이동평균선으로부터 얼마나 떨어져 있는지를 백분율로 나타낸 지표

**계산 방법**:
```
이격도 (%) = (현재 주가 / 이동평균선 가격) × 100
```

**구체적 예시**:

**예시 1 - 삼성전자**
- 현재 주가: 77,000원
- 50일 이동평균: 70,000원
- 50일 이격도 = (77,000 / 70,000) × 100 = **110%**
- 해석: 이평선 대비 10% 위에 있음 → 단기 과열 가능성

**예시 2 - 코스피 지수**
- 현재 지수: 2,400
- 200일 이동평균: 2,600
- 200일 이격도 = (2,400 / 2,600) × 100 = **92.3%**
- 해석: 이평선 대비 7.7% 아래 → 중장기 과매도 영역

**판단 기준**:
- **105% 이상**: 과열, 단기 조정 가능성 (차익실현 고려)
- **95~105%**: 정상 범위
- **95% 이하**: 과매도, 반등 가능성 (저점매수 고려)
- **110% 이상 또는 90% 이하**: 극단적 이격 → 회귀 압력 강함

**50일 vs 200일 이격도**:
- **50일 이격도**: 단기 과열/과매도 판단 → 스윙 트레이딩에 활용
- **200일 이격도**: 중장기 추세 이탈 정도 → 장기 투자 진입 타이밍

**실전 활용**:
1. 50일 이격도 115% + RSI 70 이상 → 단기 매도 신호
2. 200일 이격도 85% + VIX 급등 → 장기 매수 기회
3. 이격도가 극단에서 회귀할 때 추세 전환 시그널

**확인 방법**:
- 트레이딩뷰, 네이버 증권 차트에서 수동 계산
- 키움증권, 이베스트 HTS: 이격도 지표 기본 제공

### 2.4 RSI (Relative Strength Index)
- **의미**: 가격 상승폭과 하락폭의 상대적 강도를 나타내는 모멘텀 지표 (0~100)
- **활용**: 70 이상 과매수, 30 이하 과매도 신호
- **모니터링**: 다이버전스(주가와 RSI 방향 불일치) 시 추세 전환 암시

### 2.5 M자/W자 패턴
- **의미**: 차트상 고점/저점이 M자(하락 반전) 또는 W자(상승 반전) 형태
- **활용**: 이중천정(M) 또는 이중바닥(W) 패턴으로 추세 전환 예측
- **모니터링**: 패턴 완성 후 목표가 = 패턴 높이만큼 이동

### 2.6 CUP & Handle 패턴
- **의미**: 컵 모양 조정 후 손잡이 형태의 짧은 조정을 거쳐 상승하는 패턴
- **활용**: 강력한 상승 신호로, 손잡이 돌파 시 매수 타이밍
- **모니터링**: 컵 깊이가 30~50% 조정, 손잡이는 짧고 얕을수록 이상적

## 3. 적정주가(실적) 지표

기업의 실적 대비 주가가 적정한지 평가하는 밸류에이션 지표

### 3.1 PER (Price to Earnings Ratio, 주가수익비율)

**의미**: 현재 주가가 1주당 순이익의 몇 배인지를 나타내는 밸류에이션 지표
(P/E와 PER은 같은 용어입니다)

**계산 방법**:
```
PER = 주가 / 주당순이익(EPS)
또는
PER = 시가총액 / 당기순이익
```

**구체적 예시**:

**예시 1 - 삼성전자 (2024년 기준)**
- 현재 주가: 75,000원
- 연간 EPS(주당순이익): 5,000원
- PER = 75,000 / 5,000 = **15배**
- 해석: 현재 가격으로 매수 시, 15년간 순이익으로 투자금 회수 가능

**예시 2 - 네이버**
- 시가총액: 50조원
- 당기순이익: 2조원
- PER = 50조 / 2조 = **25배**
- 해석: 삼성전자(15배)보다 고평가 상태 (하지만 성장률이 높다면 정당화 가능)

**업종별 평균 PER (참고)**:
- IT/플랫폼: 20~40배 (성장주)
- 금융/은행: 5~10배 (가치주)
- 제조업: 10~15배
- 바이오/제약: 적자 많아 PER 무의미한 경우 많음

**판단 기준**:
- **PER 10 이하**: 저평가 가능성 (단, 실적 악화 리스크 확인 필요)
- **PER 10~20**: 적정 밸류에이션
- **PER 20~30**: 고평가 (성장주라면 정당화 가능)
- **PER 30 이상**: 심한 고평가 또는 미래 성장 기대감 반영

**종류**:
1. **Trailing PER**: 과거 12개월 실적 기준 (현재 대부분이 이것)
2. **Forward PER**: 향후 12개월 예상 실적 기준 (더 중요!)
   ```
   Forward PER = 현재 주가 / 내년 예상 EPS
   ```

**실전 활용**:

**사례 1 - 엔비디아 (2023년)**
- Trailing PER: 220배 (매우 고평가처럼 보임)
- Forward PER: 35배 (AI 수요로 폭발적 성장 예상)
- 결과: Forward PER 기준으로 매수한 투자자들이 큰 수익

**사례 2 - 동종 업체 비교**
- 삼성전자 PER: 12배
- SK하이닉스 PER: 20배
- 마이크론 PER: 18배
→ 삼성전자가 상대적으로 저평가 (단, 반도체 사이클 고려 필요)

**주의사항**:
- 적자 기업은 PER 계산 불가 (음수)
- 일회성 손익으로 왜곡 가능 → 영업이익 기반 지표도 함께 확인
- 업종 특성 무시한 절대값 비교는 무의미

**확인 방법**:
- 네이버 증권, 인베스팅닷컴: 자동 계산 제공
- 증권사 리포트: Forward PER + 목표주가 함께 제시

### 3.2 PEG (Price/Earnings to Growth)
- **의미**: P/E ÷ 예상 이익성장률
- **활용**: PEG < 1이면 성장 대비 저평가, > 1이면 고평가
- **모니터링**: 성장주 평가 시 P/E보다 유용

## 4. 역발상(위기/기회) 지표

시장 공포/탐욕 심리를 역으로 활용하는 지표

### 4.1 낙관/비관지수
- **의미**: 투자자 심리조사 결과 (Bull/Bear Ratio)
- **활용**: 극단적 낙관 시 조정 가능성, 극단적 비관 시 저점 가능성
- **모니터링**: AAII Sentiment Survey, Fear & Greed Index

### 4.2 Put/Call 지표 (Put-Call Ratio)
- **의미**: 풋옵션 거래량 ÷ 콜옵션 거래량
- **활용**: 비율 높으면 하락 베팅 많음(역발상 매수 신호), 낮으면 상승 베팅 많음(조정 가능성)
- **모니터링**: 통상 0.7 이하면 과열, 1.0 이상이면 공포
- **데이터 수집**: Massive API의 Option Chain Snapshot (15분 지연 무료), CBOE 공식 데이터
- **참고**: [[API 데이터 수집 가이드#4.2 Put/Call 지표]] - API 수집 방법 및 계산 코드

### 4.3 Margin Debt (신용융자 잔고)
- **의미**: 투자자들이 빌린 돈으로 투자한 총액
- **활용**: 급증 시 과열 신호, 급감 시 청산 압력으로 하락 가속
- **모니터링**: FINRA 월간 데이터, 전고점 대비 수준 확인

### 4.4 하이일드 스프레드 (HY Spread, High Yield Bond Spread)

**의미**: 고위험 회사채(정크본드)와 안전자산(미국 국채) 간의 금리 차이. 시장의 위험 선호도를 나타내는 핵심 지표

**계산 방법**:
```
HY 스프레드 = 하이일드 채권 수익률 - 미국 10년물 국채 수익률
```

**구체적 예시**:

**예시 1 - 정상 시장 (2024년 초)**
- 하이일드 채권 평균 수익률: 8.0%
- 미국 10년물 국채 수익률: 4.0%
- HY 스프레드 = 8.0% - 4.0% = **4.0% (400bp)**
- 해석: 위험 프리미엄이 적정 수준, 시장 안정적

**예시 2 - 위기 시장 (2020년 3월, 코로나)**
- 하이일드 채권 수익률: 11.0%
- 미국 10년물 국채 수익률: 0.7%
- HY 스프레드 = 11.0% - 0.7% = **10.3% (1,030bp)**
- 해석: 극심한 위험회피, 부도 우려 급등 → 주식 대폭락

**예시 3 - 호황 시장 (2021년)**
- 하이일드 채권 수익률: 5.0%
- 미국 10년물 국채 수익률: 1.5%
- HY 스프레드 = 5.0% - 1.5% = **3.5% (350bp)**
- 해석: 위험 선호 강함, 부도 우려 낮음 → 주식 강세

**판단 기준 (미국 기준)**:
- **300bp 이하**: 극단적 위험선호 → 과열 경계, 조정 가능성
- **300~500bp**: 정상 범위 → 안정적 시장
- **500~700bp**: 불안 시작 → 주의 필요
- **700bp 이상**: 위기 모드 → 주식 약세, 부도 우려 급등
- **1,000bp 이상**: 극심한 공포 → **역발상 매수 기회**

**왜 중요한가?**:
하이일드 채권은 주식과 강한 상관관계:
- 기업이 망하면 → 채권도 주식도 둘 다 휴지조각
- HY 스프레드 확대 = 신용 경색 = 기업 자금 조달 어려움 = 실적 악화 = 주가 하락

**실전 활용**:

**시나리오 1 - 2022년 10월**
- HY 스프레드: 550bp (고점)
- VIX: 32 (공포)
- S&P 500: 3,600 (저점)
→ **역발상 매수 타이밍** → 이후 2023년 랠리

**시나리오 2 - 2021년 말**
- HY 스프레드: 320bp (매우 낮음)
- VIX: 16 (안정)
- S&P 500: 4,800 (고점)
→ **위험 선호 과열** → 2022년 조정 시작

**상관관계**:
- HY 스프레드 ↑ (확대) → 주식 ↓
- HY 스프레드 ↓ (축소) → 주식 ↑

**확인 방법**:

1. **ETF 가격으로 간접 확인**:
   - HYG (iShares iBoxx High Yield Corporate Bond ETF)
   - JNK (SPDR Bloomberg High Yield Bond ETF)
   - 가격 하락 = 수익률 상승 = 스프레드 확대 = 위험 회피

2. **직접 데이터**:
   - FRED (세인트루이스 연은): BAMLH0A0HYM2
   - 블룸버그: H0A0 Index
   - 트레이딩뷰: TVC:US10Y와 비교

3. **한국 시장**:
   - 국내는 HY 시장 작음
   - 회사채(AA-) 스프레드로 대체 가능
   - KR 회사채 BBB- 수익률 - 국고채 3년물 수익률

**투자 전략**:
- HY 스프레드 700bp 돌파 시 → 주식 단계적 비중 축소
- HY 스프레드 1,000bp 돌파 시 → 공포 매수 준비 (VIX, Put/Call과 함께 확인)
- HY 스프레드 300bp 하회 시 → 과열 경계, 수익 일부 실현

### 4.5 공포지수 (VIX, Volatility Index)
- **의미**: S&P 500 옵션 내재변동성으로 계산한 시장 불안도
- **활용**: VIX 20 이상이면 불안, 30 이상이면 공포(역발상 매수 기회), 15 이하면 안정
- **모니터링**: VIX 급등 시 단기 바닥 신호일 수 있음

### 4.6 언론/댓글지표
- **의미**: 뉴스 헤드라인, SNS, 커뮤니티 댓글의 감성 분석
- **활용**: 극단적 긍정 시 과열(차익실현 고려), 극단적 부정 시 저점 근접 가능
- **모니터링**: 정성적 판단 필요, 대중이 열광할 때 경계

---

## 사용 가이드

1. **유동성**: 거시경제 환경 파악 → 투자 비중 결정
2. **차트**: 매매 타이밍과 추세 확인 → 진입/청산 시점 판단
3. **실적**: 종목 선정과 적정가 평가 → 매수 후보 선별
4. **역발상**: 극단적 심리 포착 → 위기를 기회로 전환

**종합 판단**: 여러 지표를 함께 고려하여 의사결정의 확신도를 높인다.
